# Image Directory Structure

Organized image assets for Twin Cape website.

## Directories

### /logo
Logo and branding assets
- `logo_text.png` - Main logo with text (displayed in header)
- `logo_icon.png` - Icon-only version (optional)
- `favicon.png` - Favicon

### /hero
Full-width hero section images
- `homepage.jpg` - Homepage hero banner background
- `services.jpg` - Services page hero (optional)
- Other section hero images

### /content
Regular page content images
- Product screenshots
- Team photos
- Infographics
- Illustrations
- Other inline content images

## Best Practices

- Use descriptive filenames
- Optimize images for web (use tools like ImageOptim or TinyPNG)
- Hero images: typically 1200x600px or larger (landscape)
- Logos: keep padding around the image for flexibility
- All images should have proper `alt` text in HTML for accessibility
